transform : 요소를 여러가지 방법으로 변형

  1. translate : 위치이동
    translateX / translateY
  2. scale : 크기변형
    글자랑 사진도 같이 억지로 막막 구겨서 바꾸기 떄문에 잘 사용하지는 않음
    scaleX / scaleY
  3. rotate : 회전
    rotateX / rotateY
  4. skew : 뒤틀기
    잘 사용하지 않음 skewX / skewY
