animation : @keyframes를 이용해 애니메이션을 만든다.

  1. animation-name : @keyframes 이름
  2. animation-duration : 애니메이션이 진행될 시간(s)
  3. animation-timing-function : 애니메이션의 속도 변화 유형
    • ease : 느린 시작으로 빨라진 뒤 느린 끝 (기본값)
    • linear : 처음부터 끝까지 일정한 속도
    • ease-in : 느린 시작
    • ease-out : 느린 끝
    • ease-in-out : 느린 시작과 끝
  4. animation-delay : 애니메이션을 시작하기 전 지연시간(s)
  5. animation-iteration-count : 애니메이션의 재생횟수를 지정(N, infinite)
  6. animation-direction : 애니메이션이 종료된 후 반복할 떄 진행하는 방향
    • normal : 기본값, 0% > 100%
    • reverse : 역방향, 100% > 0%
    • alternate : 정방향으로 움직이고 역방향으로 다시감, 0% > 100% > 0%
    • alternate-reverse : 역방향으로 움직이고 정방향으로 다시감, 100% > 0% >100%
    • 단, alternate는 iteration-count를 2 이상으로 해야 정방향, 역방향 모두 볼 수 있다.
  7. animation-fill-mode : 애니메이션 미실행시(시작할 때 말고, 종료 / 대기)요소의 스타일
    • none : 기본값, 애니메이션이 끝난 후 상태를 지정하지 않음. 그냥 끝
    • forwards : 애니메이션이 끝난 후 그 지점 그대로 있음. 끝난 상태 그대로 끝남.
    • backwards : 애니메이션이 끝난 후 시작점으로 돌아옴
    • both : 애니메이션의 앞뒤 결과를 조합해서 설정
    • inherit : 애니메이션의 상태를 상위요소에 상속.
  8. animation-paly-state : 애니메이션의 재생상태(재생 / 중지)를 지정
  9. animation : (name) (duration) (timing-function) (delay) (iteration-count) (direction); ()는 구분을 위해 작성, 중간에 필요 없을 경우 생략
