CSS 스타일시트 연결하기

  • 인라인 스타일

    태그 옆에(열리는 태그 안에) style 속성으로 적어주는 방법

    style은 속성과 속성값이 있다.
    style="속성: 속성값;" 의 형식으로 사용한다.

    box1

  • 내부 스타일시트

    css를 html 문서 내부에 스타일 태그로 적는 방법

    head 태그 안에서 style 태그로 작성한다.

    태그를 쓰고, {속성: 속성값;} 으로 작성한다.

  • 외부 스타일시트

    스타일 속성들을 css 파일로 따로 저장해서 html 문서에 링크 시키는 방법

    보통의 홈페이지들은 이 방식을 사용하여 제작한다.

    홈페이지가 커지면 하나하나 스타일을 줄 수 없고,
    일일이 하나하나 넣다보면 스크롤이 엄청나게 길어져서 나중에 찾기가 어려워진다.
    통상은 외부스타일 시트를 css 파일을 만들어서 따로 저장해서 html 문서에 링크시켜서 사용한다.

    1. 컨트롤 + N(기타 다른 방법을 써도 상관 없음) 으로 새창을 열고
    2. css 확장자명의 파일을 저장한 후
    3. 여기에 태그 {속성: 속성값;}으로 스타일 속성값을 작성한다.
    4. 한가지씩 아니고, 한번에 여러가지 속성도 넣을 수 있다.
    5. html 문서 안에 head 태그에서 link 태그에 href 속성으로 경로를 연결해서 사용한다.
