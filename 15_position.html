position : 요소의 위치 지정방법 유형을 지정

<0l>
  • static : 원래 상태
  • relative :absolute 영역의 기준
  • absolute : 상위 요소의 relative을 기준으로 위치를 재배치
    relative가 상위요소(형제태그 아니고, 부모 또는 조상님태그)에 위치하고, absolute가 하위요소(형제태그 아니고 자식 또는 자손태그)에 위치한다. 더 가까운 상위 요소에 position 속성값을 갖니 요소가 있따면, 그 요소를 기준으로 배치된다.
  • fixed :요소의 위치를 스크린을 기준으로 배치
  • sticky: 필수적으로 top, bottom, left, roght 중에 하나를 설정해 주어야하고, 설정된 위치에 도달하기 전 까지는 static 속성처럼 행동하다가 설정된 위치에 다닫르면, fixed처럼 행동한다.
  • 같이 사용되는 스타일 속성 : top or bottom, left or right(위에서 얼마나 올껀지, 아래에서 얼마나 올껀지, 왼쪽에서 얼마나 올껀지, 오른쪽에서 얼마나 올껀지)
    단위 : px, %, auto(줬던 값을 무효화 시킴, 0을 주면 왼쪽 상단으로 오는데 그게 아니고 그냥 무효화 시키는 것)
  • span 태그나 a 태그 같은 인라인 요소에 absolute 값을 주게 되면 블락 속성을 가지게 된다.
