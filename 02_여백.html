margin / padding 속기법

margin이나 padding에 값을 1개 초과해서 주면, 값을 주는 만큼 해당되는 방향으로 여백이 생긴다.

margin / padding : Npx; → 사방으로 Npx 만큼 여백이 들어옴

margin / padding : Npx Npx; → 위/아래 왼/오른

margin / padding : Npx Npx Npx; → 위 왼/오른 아래

margin / padding : Npx Npx Npx Npx; → 위 오른 아래 왼

margin: 0 auto

너비값이 정해진 요소를 화면상(띄워진 창의 크기에서 ..) 또는 내 부모요소에서 중간정렬을 하기 위한 값

매번 너비가 바뀌므로 정확하게 수치상으로 계산하는 것은 비효율적이다. 반응형의 기초! 많이 사용되어진다.

글자만 중간에 보내고 싶다고 박스 안만들고 그냥 하면 안됩니다. 글자 정렬은 다른게 있으므로, 무조건 박스를 만들어서 기본 너비값을 넣어주고 사용해야한다.

나중에 적은 여백값이 더 우선순위로 적용된다.

위에서 언급한 속기법이랑 비슷한 느낌으로 사용한다. margin: Npx auto; 라고 입력하면, 위아래 N픽셀만큼, 좌우는 정렬된다.
