link 태그로 연결해서 사용

html 파일에 사용

  1. 구글 폰트 검색 > 한국어
  2. 원하는 글자체 선택
  3. + select this style
  4. link 부분 복사후 head 태그 안에 붙여넣기
  5. CSS rules families 부분에서 복사해서 head 태그 > style 태그 안에 적용할 태그에 넣기



@import로 연결해서 사용

스타일시트에 사용(외부, 내부 둘다 가능)

  1. 구글 폰트 검색 > 한국어
  2. 원하는 글자체 선택
  3. + select this style
  4. @import 부분 복사후 스타일 시트안에 붙여넣기
  5. CSS rules families 부분에서 복사해서 적용할 태그에 넣기
