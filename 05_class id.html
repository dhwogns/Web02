class : 태그에 이름을 줄 때 사용한다. 여러개의 태그에 같은 이름을 사용할 수 있다. 띄워쓰기로 하나의 태그에 여러개의 클래스 이름을 사용할 수 있다.

id : 태그에 이름을 줄 때 사용한다. 여러개의 태그에 같은 이름을 사용할 수 없다.(유일한 이름이 되어야 한다.) 아이디와 클래스는 이름이 중복되어도 사용할 수 있다.

클래스나 아이디의 값은 영어로 써야한다. 대/소문자 구분 해야한다. 시작자리에 기호는 ""밖에 못 들어간다. 기호는 영어 먼저 쓰고난 다음에 -(대시)나 ~(물결)이나 _(언더바) 같은 것들을 사용한다. 가급적이면 내용과 관련있는 영문명으로 사용하는게 좋다.

해드태그의 타이틀 밑에서 시작한다. {} 중괄호를 사용한다. 클래스는 . 아이디는 # 으로 불러온다. 나중에 작성한 값이 우선순위로 적용된다. class 보다 id가 우선순위로 적용된다.
