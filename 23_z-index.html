z-index : 요소의 겹치는 순서를 지정

음수, 양수 모두 사용이 가능하다. 숫자가 높을수록 겹쳤을 때 더 위쪽에 나타난다.

반드시 position과 함께 사용한다.(absolute 든 relative 든 간에 무조건 같이 써야한다.

position을 쓰지 않고 그냥 z-index만 쓰면 적용되지 않는다.

같은 순서값을 가진 요소는 더 나중에 작성된 요소가 더 위에 나타난다.

형제태크가 아니더라도 position 값을 가지 요소 끼리는 z-index 값이 적용된다.

'부모요소가 자식요소보다 앞으로 나온다'라는 것은 말이 안된다.

요소의 위치값이 변경되지 않아야 할경우 position: relative 값을 줘서 z-index와 사용해주면 된다.
